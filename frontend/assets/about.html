Etymoscope is a small tool used to visualize relationships among etymologies. Two words have a link between them if and only if one of the words is present in the Related Words section of the other word's page on The Online Etymology Dictionary.

Discovered a bug? Interested in contributing? View the source code on GitHub or contact the creator.
